答客問


  • 演講議程和資料分析上手課程只能擇一參加嗎？

    本年會採三軌並行制。年會全體活動及演講議程在國際會議廳舉行，資料分析上手課程在第二會議室舉行，持一般票（非 g0v 黑客松票）者可自由進出所有會議室，參加任一活動。


  • 參加資料分析上手課程需要有什麼必備知識嗎？

    本年會的資料分析上手課程適合沒有程式與統計基礎的同好參與，若熟悉任一程式語言（如 C 語言或 PHP 語言）更佳。參加者請自備筆記型電腦以在課堂上即時做練習。年會前一個星期我們將在網站上提供套件清單、投影片、虛擬機器和課程專用的設定 script 開放下載，讓參加者可以事先準備。


  • 想請問 g0v 黑客松的進行方式？

    可參考這頁 g0v 黑客松 說明。這次兩天黑客松稍微不同，第一天在年會主題演講後黑客松開始提案與自介，當場組隊開始 hack，第二天下午五點在年會成果報告。完全歡迎偷跑！不需經驗也無須作品，但歡迎上 g0v IRC 交流。自由黑客工作區連結在 http://hack.g0v.tw/。
